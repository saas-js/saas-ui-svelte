# Format Byte

## Used to format bytes to a human-readable format.

<Canvas of={FormatByteStories.Basic} />

## Usage

```tsx
<script>
	import { FormatByte } from "$saas/utilities/format-byte";
	import { Text } from "$saas/typography/text";
</script>

<Text>
	File size: <FormatByte value={1000} />
</Text>
```

## Examples

### Sizes

The format function works for any size of bytes.
