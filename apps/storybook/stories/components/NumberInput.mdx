# Number Input

## Used to enter a number, and increment or decrement the value using stepper buttons.

[Ark UI](https://ark-ui.com/docs/components/number-input)

<Canvas of={NumberInputStories.Basic} />

## Anatomy

```tsx
<script>
	import { NumberInput } from "$saas/components/number-input";
</script>

<NumberInput name="number" />
```

## Examples

### Sizes

Use the `size` prop to change the size of the number input. Available sizes are `xs`, `sm`, `md` (default), and `lg`.

<Canvas of={NumberInputStories.Sizes} />

### Min and Max

Use the `min` and `max` props to set the minimum and maximum values of the number input. If value entered is less than min or greater than max, the value will be clamped to the nearest boundary on blur, or enter key press.

<Canvas of={NumberInputStories.MinAndMax} />

### Step

Use the `step` prop to change the increment or decrement interval of the number input.

<Canvas of={NumberInputStories.Step} />

### Mouse Wheel

Use the `allowMouseWheel` prop to enable the mouse wheel to change the value.

<Canvas of={NumberInputStories.MouseWheel} />

### Disabled

Use the `disabled` prop to disable the number input.

<Canvas of={NumberInputStories.Disabled} />

### Invalid

Use the `Field` component and pass the `invalid` prop to indicate that the number input is invalid.

<Canvas of={NumberInputStories.Invalid} />

### Helper Text

Use the `Field` component to add a label and helper text to the number input.

<Canvas of={NumberInputStories.WithField} />

### Formatting

Use the `formatOptions` prop to format the number input value. The value maps to `Intl.NumberFormatOptions` and is applied based on the current locale.
