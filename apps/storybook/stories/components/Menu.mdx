# Menu

## Used to create an accessible dropdown menu.

[Ark UI](https://ark-ui.com/docs/components/menu)

<Canvas of={MenuStories.Basic} />

## Anatomy

```tsx
<script>
	import { Menu } from "$saas/components/menu";
</script>

<Menu.Root>
	<Menu.Button />
	<Menu.Content>
		<Menu.Item value="item" />
	</Menu.Content>
</Menu.Root>
```
