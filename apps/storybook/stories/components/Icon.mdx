# Icon

## Used to display an SVG icon

<Canvas of={IconStories.Basic} />

## Anatomy

```tsx
<script>
	import { Icon } from "$saas/icon";
	import Heart from "phosphor-svelte/lib/Heart";
</script>

<Icon as="{Heart}" weight="regular" />
```

You can also use raw SVG paths by passing them as children:

```tsx
<Icon viewBox="0 0 24 24">
	<path d="..." />
</Icon>
```

## Examples

### Sizes

Use the `size` prop to change the icon dimensions. You can set the value to `xs`, `sm`, `md`, `lg`, `xl`, or `2xl`.
