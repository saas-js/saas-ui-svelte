# ToggleTip

## Looks like a tooltip, but works like a popover.

[Ark UI](https://ark-ui.com/docs/components/popover)

<Canvas of={ToggleTipStories.Basic} />

## Anatomy

```tsx
<script>
	import { ToggleTip, InfoTip } from "$saas/components/toggle-tip";
</script>

<ToggleTip content="...">
	<button />
</ToggleTip>
```

## Examples

### Info Tip

Use the `InfoTip` component to display an info tip. This component renders an icon button with an info icon by default.

Useful for landing pages to display additional information about a feature.

<Canvas of={ToggleTipStories.InfoTip} />

### Sizes

Use the `size` prop to change the size of the toggle tip content. Available sizes are `xs`, `sm`, `md` (default), and `lg`.

<Canvas of={ToggleTipStories.Sizes} />

### Positioning

Use the `positioning` prop to control where the toggle tip appears relative to its trigger.

<Canvas of={ToggleTipStories.Positioning} />

### Custom Content

You can pass a snippet to the `content` prop for custom HTML content.

<Canvas of={ToggleTipStories.CustomContent} />
