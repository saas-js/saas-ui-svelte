# Slider

## Used to allow users to make selections from a range of values.

[Ark UI](https://ark-ui.com/docs/components/slider)

<Canvas of={SliderStories.Basic} />

## Anatomy

### Simple Usage

```tsx
<script>
	import { Slider } from "$saas/components/slider";
</script>

<Slider.Root defaultValue={[40]}>
	<Slider.Control>
		<Slider.Track>
			<Slider.Range />
		</Slider.Track>
		<Slider.Thumb index={0} />
	</Slider.Control>
</Slider.Root>
```

## Examples

### Sizes

Use the `size` prop to change the size of the slider. You can set the value to `sm`, `md`, or `lg`.

<Canvas of={SliderStories.Sizes} />

### Variants

Use the `variant` prop to change the visual style of the slider. Options are `outline` (default) and `solid`.

<Canvas of={SliderStories.Variants} />

### Colours

Use the `colour` prop to change the colour scheme of the slider. It supports all standard colour palettes.

<Canvas of={SliderStories.Colours} />

### Label

Use the `Slider.Label` component to add a label to the slider.

<Canvas of={SliderStories.Label} />

### Range Slider

Set the `defaultValue` prop to an array with two values to create a range slider with two thumbs.

<Canvas of={SliderStories.Range} />

### Disabled

Use the `disabled` prop to disable the slider.

<Canvas of={SliderStories.Disabled} />

### Marks

Use `Slider.MarkerGroup` and `Slider.Marker` components to add markers to the slider track.

<Canvas of={SliderStories.Marks} />

### Vertical

Use the `orientation` prop set to `vertical` to create a vertical slider.

<Canvas of={SliderStories.Vertical} />

### Controlled

Use the `value` and `onValueChange` props to create a controlled slider.

<Canvas of={SliderStories.Controlled} />

### Steps

Use the `step` prop to set the step value of the slider.

<Canvas of={SliderStories.Steps} />

### Change End

Use the `onValueChangeEnd` prop to listen to the end of the slider change. This fires when the user releases the thumb, as opposed to `onValueChange` which fires continuously during dragging.

<Canvas of={SliderStories.ChangeEnd} />
