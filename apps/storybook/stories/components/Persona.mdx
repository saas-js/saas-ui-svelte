# Persona

## Used to represent a user or entity with avatar, name, and additional details.

[Ark UI](https://ark-ui.com/docs/components/avatar)

<Canvas of={PersonaStories.Basic} />

## Anatomy

```tsx
<script>
	import { Persona } from "$saas/components/persona";
</script>

<Persona.Root>
	<Persona.Avatar>
		<Persona.PresenceBadge />
	</Persona.Avatar>
	<Persona.Details>
		<Persona.Label />
		<Persona.SecondaryLabel />
		<Persona.TertiaryLabel />
	</Persona.Details>
</Persona.Root>
```

## Examples

### Sizes

Use the `size` prop to change the size of the persona.

<Canvas of={PersonaStories.Sizes} />

### Shape

Use the `shape` prop on the Avatar to change the shape of the avatar, from rounded to square.

<Canvas of={PersonaStories.Shapes} />

### Presence

Use the `PresenceBadge` component inside the Avatar to show the user's presence status.

<Canvas of={PersonaStories.Presence} />

### Out of Office

Use the `TertiaryLabel` to show additional information such as out of office status.

<Canvas of={PersonaStories.OutOfOffice} />

### Ring

Use the `ring` prop on the Avatar to add a ring around the avatar.

<Canvas of={PersonaStories.WithRing} />

### Avatar Only

The Persona component can be used with just an avatar and presence badge.

<Canvas of={PersonaStories.AvatarOnly} />
