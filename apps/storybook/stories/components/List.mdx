# List

## Used to display a list of items.

<Canvas of={ListStories.Basic} />

## Anatomy

```tsx
<script>
	import { List } from "$saas/components/list";
</script>

<List.Root>
	<List.Item>Item 1</List.Item>
	<List.Item>Item 2</List.Item>
</List.Root>
```

## Examples

### Ordered

Pass the `as="ol"` prop to create an ordered list.

<Canvas of={ListStories.Ordered} />

### Icon

Use the `List.Indicator` component to add an icon to the list. When using indicators, set `variant="plain"` on the root to remove default list markers.
