# Action Bar

## Used to display a floating action bar at the bottom of the screen with a set of bulk actions. Typically shown when items are selected in a list or table.

<Canvas of={ActionBarStories.Basic} />

## Anatomy

```tsx
<script>
	import {
		ActionBar,
		ActionBarSelectionTrigger,
		ActionBarSeparator,
		ActionBarCloseButton,
	} from "$saas/components/action-bar";
	import { Button } from "$saas/components/button";
</script>

<ActionBar open={selectedCount > 0}>
	<ActionBarSelectionTrigger>{selectedCount} selected</ActionBarSelectionTrigger>
	<ActionBarSeparator />
	<Button variant="outline" size="sm">Delete</Button>
	<Button variant="outline" size="sm">Share</Button>
	<ActionBarCloseButton />
</ActionBar>
```

## Examples

### With Close Trigger

Render the `ActionBarCloseButton` to allow users to dismiss the action bar. Pass the `onOpenChange` handler to control the visibility.

<Canvas of={ActionBarStories.Closable} />

### Multiple Actions

Action bars can contain multiple action buttons. Use the `ActionBarSeparator` to visually group related actions.

<Canvas of={ActionBarStories.MultipleActions} />

### Interactive Selection

The action bar is designed to work with table or checkbox selections. It automatically appears when items are selected and can be dismissed.

<Canvas of={ActionBarStories.Interactive} />
