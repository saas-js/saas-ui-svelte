# Progress

## Used to display the progress status for a task.

[Ark UI](https://ark-ui.com/docs/components/progress-linear)

<Canvas of={ProgressStories.Basic} />

## Anatomy

### Simple Usage

```tsx
<script>
	import { Progress } from "$saas/components/progress";
</script>

<Progress.Root value={50}>
	<Progress.Track>
		<Progress.Range />
	</Progress.Track>
</Progress.Root>
```

## Examples

### Sizes

Use the `size` prop to change the size of the progress bar. You can set the value to `xs`, `sm`, `md`, `lg`, or `xl`.

<Canvas of={ProgressStories.Sizes} />

### Variants

Use the `variant` prop to change the visual style of the progress bar.

<Canvas of={ProgressStories.Variants} />

### Shapes

Use the `shape` prop to control the border radius of the progress bar.

<Canvas of={ProgressStories.Shapes} />

### Colours

Use the `colour` prop to change the colour scheme of the component. It supports all standard colour palettes.

<Canvas of={ProgressStories.Colours} />

### With Label

Use the `Progress.Label` component to add a label to the progress bar.

<Canvas of={ProgressStories.WithLabel} />

### Inline Label

Compose the `Progress.Label` and `Progress.ValueText` components with a layout component to create an inline label for the progress bar.

<Canvas of={ProgressStories.InlineLabel} />

### Info Tip

Use the `info` prop on `Progress.Label` to add a tooltip with helpful context.

<Canvas of={ProgressStories.WithInfoTip} />

### Indeterminate

Set the `value` prop to `null` to show an indeterminate progress bar. This is useful when the progress is unknown.

<Canvas of={ProgressStories.Indeterminate} />

### Stripes

Set the `striped` prop to `true` to add stripes to the progress bar.

<Canvas of={ProgressStories.Stripes} />

### Animated Stripes

Set the `animated` prop to `true` to animate the stripes.

<Canvas of={ProgressStories.AnimatedStripes} />

### Values

Display different progress values to show completion status.

<Canvas of={ProgressStories.Values} />
