# Stat

## Used to display a statistic with a title and value.

<Canvas of={StatStories.Basic} />

## Anatomy

```tsx
<script>
	import { Stat } from "$saas/components/stat";
</script>

<Stat.Root>
	<Stat.Label />
	<Stat.ValueText />
	<Stat.HelpText>
		<Stat.UpTrend />
		<Stat.DownTrend />
	</Stat.HelpText>
</Stat.Root>
```

## Examples

### Format Options

Pass the `formatOptions` to the `Stat.ValueText` component to format the value using `Intl.NumberFormat`.

<Canvas of={StatStories.FormatOptions} />

### Indicator

Use the `Stat.DownTrend` or `Stat.UpTrend` components to display a trend indicator with an arrow.

<Canvas of={StatStories.Indicator} />

### Value Unit

Use the `Stat.ValueUnit` component to display a unit alongside the value.

<Canvas of={StatStories.ValueUnit} />

### Progress Bar

Combine with the `Progress` component to show a statistic with a progress bar.

<Canvas of={StatStories.ProgressBar} />

### Icon

Add an icon alongside the label using the `Icon` component.

<Canvas of={StatStories.Icon} />

### Trend

Use the `Stat.UpTrend` component with the default `subtle` variant to show a trend badge.

<Canvas of={StatStories.Trend} />

### Colours

The trend indicators use the `green` and `red` colour palettes by default.

<Canvas of={StatStories.Colours} />
