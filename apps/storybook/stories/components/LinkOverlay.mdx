# Link Overlay

## Used to stretch a link over a container.

The LinkOverlay component provides a semantic way to link an entire component or card.

<Canvas of={LinkOverlayStories.Basic} />

## Anatomy

```tsx
<script>
	import { LinkBox, LinkOverlay } from "$saas/components/link-overlay";
</script>

<LinkBox>
	<LinkOverlay href="#">Link text</LinkOverlay>
</LinkBox>
```

## Examples

### Article

Here's an example of using LinkOverlay to link an entire article. When LinkOverlay is inside a heading, use the default `variant="none"` so the text inherits the heading styles.

Inner links that need to be clickable above the overlay require `relative z-1` classes.

<Canvas of={LinkOverlayStories.Article} />
