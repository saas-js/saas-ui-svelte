# Segment Group

## Used to pick one choice from a linear set of options.

[Ark UI](https://ark-ui.com/docs/components/segment-group)

<Canvas of={SegmentGroupStories.Basic} />

## Anatomy

A Segment Group (also known as Segmented Control) allows users to choose one option from a set of mutually exclusive options. It's commonly used for filtering content or switching between views.

```tsx
<script>
	import { SegmentGroup } from "$saas/components/segment-group";
</script>

<SegmentGroup.Root defaultValue="react">
	<SegmentGroup.Indicator />
	<SegmentGroup.Item value="react" label="React" />
	<SegmentGroup.Item value="vue" label="Vue" />
	<SegmentGroup.Item value="svelte" label="Svelte" />
</SegmentGroup.Root>
```
