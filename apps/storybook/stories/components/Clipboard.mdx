# Clipboard

## Used to copy text to the clipboard.

[Ark UI](https://ark-ui.com/docs/components/clipboard)

<Canvas of={ClipboardStories.Basic} />

## Anatomy

```tsx
<script>
	import {
		Clipboard,
		ClipboardButton,
		ClipboardIconButton,
		ClipboardInput,
	} from "$saas/components/clipboard";
</script>

<Clipboard value="...">
	<ClipboardButton />
	<ClipboardIconButton />
	<ClipboardInput />
</Clipboard>
```
