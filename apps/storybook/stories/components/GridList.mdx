# Grid List

## Used to display a list of interactive items with a grid-based layout.

<Canvas of={GridListStories.Basic} />

## Anatomy

```tsx
<script>
	import { GridList } from "$saas/components/grid-list";
</script>

<GridList.Root interactive>
	<GridList.Header>Header</GridList.Header>
	<GridList.Item>
		<GridList.Cell>Cell</GridList.Cell>
	</GridList.Item>
</GridList.Root>
```
