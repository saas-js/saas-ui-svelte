# Radio Card

## Used to select an option from a list with a larger interaction surface.

[Ark UI](https://ark-ui.com/docs/components/radio-group)

<Canvas of={RadioCardStories.Basic} />

## Anatomy

A RadioCard is a form element with a larger interaction surface represented as a card.

**Simple Usage:**

```tsx
<script>
	import { RadioCard } from "$saas/components/radio-card";
</script>

<RadioCard.Root defaultValue="next">
	<RadioCard.Label>Select framework</RadioCard.Label>
	<HStack align="stretch">
		<RadioCard.Item
			label="Next.js"
			description="Best for apps"
			value="next"
		/>
		<RadioCard.Item
			label="Vite"
			description="Best for SPAs"
			value="vite"
		/>
	</HStack>
</RadioCard.Root>
```

**With Icon:**

```tsx
<script>
	import { RadioCard } from "$saas/components/radio-card";
	import ArrowRight from "phosphor-svelte/lib/ArrowRight";
</script>

<RadioCard.Root defaultValue="allow">
	<RadioCard.Label>Select permission</RadioCard.Label>
	<HStack align="stretch">
		<RadioCard.Item
			label="Allow"
			description="This user can access the system"
			value="allow"
			icon={ArrowRight}
		/>
	</HStack>
</RadioCard.Root>
```
