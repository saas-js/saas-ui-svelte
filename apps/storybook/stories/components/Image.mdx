# Image

## Used to display images with consistent styling and responsive behaviour.

<Canvas of={ImageStories.Basic} />

## Anatomy

```tsx
<script>
	import { Image } from "$saas/components/image";
</script>

<Image src="..." alt="..." />
```

## Examples

### Height

Use the `height` prop to set a fixed height for the image.

<Canvas of={ImageStories.Height} />

### Circular

Create circular images by combining `rounded="full"` with equal width and height values.

<Canvas of={ImageStories.Circular} />

### Aspect Ratio

Use the `aspectRatio` prop to maintain a consistent aspect ratio regardless of the source image dimensions.

<Canvas of={ImageStories.AspectRatio} />

### Object Fit

Use the `fit` prop to control how the image fills its container.

<Canvas of={ImageStories.ObjectFit} />

### Rounded

Use the `rounded` prop to change the border radius of the image.
