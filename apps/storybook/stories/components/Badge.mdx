# Badge

Used to highlight an item's status for quick recognition.

<Canvas of={BadgeStories.Basic} />

## Anatomy

The Badge component is a simple inline element for displaying status, labels, or counts.

```tsx
<script>
	import { Badge } from "$saas/components/badge";
</script>

<Badge>Badge</Badge>
```
