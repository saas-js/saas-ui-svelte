# Pin Input

## Used to capture a pin code or OTP from the user.

[Ark UI](https://ark-ui.com/docs/components/pin-input)

<Canvas of={PinInputStories.Basic} />

## Anatomy

```tsx
<script>
	import { PinInput } from "$saas/components/pin-input";
</script>

<PinInput />
```

## Examples

### Sizes

Use the `size` prop to change the size of the pin input.

<Canvas of={PinInputStories.Sizes} />

### Variants

Use the `variant` prop to change the visual style. Values can be `outline`, `subtle`, or `flushed`.

<Canvas of={PinInputStories.Variants} />

### One Time Code

Use the `otp` prop to make the pin input component behave like a one-time code input. This helps improve the user experience when entering OTP codes by setting `autocomplete="one-time-code"`.

<Canvas of={PinInputStories.OneTimeCode} />

### Mask

Use the `mask` prop to obscure the entered pin code, similar to a password input.

<Canvas of={PinInputStories.Mask} />

### Placeholder

<Canvas of={PinInputStories.Placeholder} />
