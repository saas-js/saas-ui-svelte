# File Upload

## Used to select and upload files from the user's device.

<Canvas of={FileUploadStories.Basic} />

## Anatomy

```tsx
<script>
	import { FileUpload } from "$saas/components/file-upload";
	import UploadSimple from "phosphor-svelte/lib/UploadSimple";
</script>

<FileUpload.Root>
	<FileUpload.Trigger>
		<UploadSimple />
		Upload file
	</FileUpload.Trigger>
	<FileUpload.List />
</FileUpload.Root>
```

### Directory
