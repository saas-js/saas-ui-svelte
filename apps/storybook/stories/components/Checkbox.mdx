# Checkbox

## Used in forms when a user needs to select multiple values from several options.

[Ark UI](https://ark-ui.com/docs/components/checkbox)

<Canvas of={CheckboxStories.Basic} />

## Anatomy

**Simple Usage:**

```tsx
<script>
	import { Checkbox } from "$saas/components/checkbox";
</script>

<Checkbox.Root label="Accept terms and conditions" />
```

**Composition API:**

```tsx
<script>
	import { Checkbox } from "$saas/components/checkbox";
	import { Check } from "phosphor-svelte";
</script>

<Checkbox.Root>
	<Checkbox.Control>
		<Checkbox.Indicator>
			<Check weight="bold" />
		</Checkbox.Indicator>
	</Checkbox.Control>
	<Checkbox.Label>Accept terms and conditions</Checkbox.Label>
	<Checkbox.HiddenInput />
</Checkbox.Root>
```
