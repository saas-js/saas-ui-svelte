### Autoresize

Use the `Textarea.Autoresize` component to make the textarea automatically expand as the user types. You can control the minimum and maximum number of rows with the `minRows` and `maxRows` props.

<Canvas of={TextareaStories.Autoresize} />

```tsx
<script>
	import { Textarea } from "$saas/components/textarea";
</script>

<Textarea.Autoresize
	placeholder="Start typing..."
	minRows="{1}"
	maxRows="{40}"
/>
```

The autoresize component uses a hidden `<pre>` element to calculate the correct height based on the content.
