# Editable

## Used for inline renaming of some text.

<Canvas of={EditableStories.Basic} />

## Anatomy

```tsx
<script>
	import { Editable } from "$saas/components/editable";
</script>

<Editable.Root defaultValue="Click to edit">
	<Editable.Area>
		<Editable.Preview />
		<Editable.Input />
	</Editable.Area>
</Editable.Root>
```
