# Checkbox Card

## Used to select or deselect options displayed within cards.

<Canvas of={CheckboxCardStories.Basic} />

## Anatomy

```tsx
<script>
	import { CheckboxCard } from "$saas/components/checkbox-card";
</script>

<CheckboxCard.Root>
	<CheckboxCard.Control>
		<CheckboxCard.Content>
			<CheckboxCard.Label>Label</CheckboxCard.Label>
			<CheckboxCard.Description>Description</CheckboxCard.Description>
		</CheckboxCard.Content>
		<CheckboxCard.Indicator />
	</CheckboxCard.Control>
</CheckboxCard.Root>
```

## Examples

### Group

Use multiple CheckboxCard components together to create a group selection. Manage the checked state externally for group behavior.
