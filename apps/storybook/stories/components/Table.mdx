# Table

## Used to display data in a tabular format.

<Canvas of={TableStories.Basic} />

## Anatomy

```tsx
<script>
	import { Table } from "$saas/components/table";
</script>

<Table.Root>
	<Table.Header>
		<Table.Row>
			<Table.ColumnHeader />
		</Table.Row>
	</Table.Header>
	<Table.Body>
		<Table.Row>
			<Table.Cell />
		</Table.Row>
	</Table.Body>
</Table.Root>
```
