# Radio

## Used to select one option from several options.

[Ark UI](https://ark-ui.com/docs/components/radio-group)

<Canvas of={RadioStories.Basic} />

## Anatomy

**Simple Usage:**

```tsx
<script>
	import { Radio } from "$saas/components/radio";
</script>

<Radio.Group defaultValue="1">
	<Radio.Item value="1" label="Option 1" />
	<Radio.Item value="2" label="Option 2" />
	<Radio.Item value="3" label="Option 3" />
</Radio.Group>
```

**Composition API:**

```tsx
<script>
	import { Radio } from "$saas/components/radio";
</script>

<Radio.Group defaultValue="react">
	<Radio.Label>Select framework</Radio.Label>
	<Radio.Item value="react">
		{#snippet children()}
			<Radio.ItemControl>
				<Radio.Indicator />
			</Radio.ItemControl>
			<Radio.ItemText>React</Radio.ItemText>
			<Radio.ItemHiddenInput />
		{/snippet}
	</Radio.Item>
</Radio.Group>
```
