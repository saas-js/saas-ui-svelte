# Field

The Field component is a wrapper that provides context to its children. It handles accessibility and state for common form field elements.

<Canvas of={FieldStories.Basic} />

## Anatomy

```tsx
<script>
	import { Field } from "$saas/components/field";
	import { Input } from "$saas/components/input";
</script>

<Field.Root>
	<Field.Label>Label</Field.Label>
	<Input />
	<Field.HelperText>Helper text</Field.HelperText>
	<Field.ErrorText>Error text</Field.ErrorText>
</Field.Root>
```
