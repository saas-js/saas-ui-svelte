# Navbar

## Top navigation component, typically used as primary navigation in websites or stacked application layouts.

<Canvas of={NavbarStories.Basic} />

## Anatomy

```tsx
<script>
	import { Navbar } from "$saas/components/navbar";
</script>

<Navbar.Root>
	<Navbar.Content>
		<Navbar.Brand>
			<!-- Logo goes here -->
		</Navbar.Brand>
		<Navbar.ItemGroup>
			<Navbar.Item>
				<Navbar.Link href="#">Home</Navbar.Link>
			</Navbar.Item>
		</Navbar.ItemGroup>
	</Navbar.Content>
</Navbar.Root>
```

## Examples

### Bordered

Add `bordered` prop to display a bottom border on the navbar.

<Canvas of={NavbarStories.Bordered} />

### Glass Variant

The `glass` variant renders the navbar with a translucent background and blur effect, ideal for overlaying content.

<Canvas of={NavbarStories.GlassVariant} />

### Solid Variant

The `solid` variant renders the navbar with a solid background colour. Use the `colour` prop to change the background colour palette.

<Canvas of={NavbarStories.SolidVariant} />

### Variants

Use the `variant` prop to change the visual style of the navbar.

<Canvas of={NavbarStories.Variants} />

### Sticky

Use `position="sticky"` to make the navbar stick to the top of the viewport when scrolling.

<Canvas of={NavbarStories.Sticky} />

### Hide on Scroll

Use the `shouldHideOnScroll` prop to hide the navbar when scrolling down and show it when scrolling up. The component automatically detects the nearest scrollable parent, or you can pass a `parentRef` to specify the scroll container.

<Canvas of={NavbarStories.HideOnScroll} />

### Colours

When using the `solid` variant, use the `colour` prop to change the background colour.

<Canvas of={NavbarStories.Colours} />

### Mobile Navigation

Add a mobile navigation menu to the navbar using the Drawer component. The desktop navigation is hidden on small screens and replaced with a hamburger menu that opens a drawer.

<Canvas of={NavbarStories.MobileNavigation} />
