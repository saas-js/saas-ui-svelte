# Box

The most abstract styling component on top of which all other components are built. Box renders a `div` by default and accepts all standard HTML attributes plus Tailwind classes.

<Canvas of={BoxStories.Basic} />

## Usage

The Box component provides an easy way to write styles with ease using Tailwind classes.

```tsx
<script>
	import { Box } from "$saas/layout/box";
</script>

<Box class="bg-red-600 p-4 text-white">
	This is the Box
</Box>
```
