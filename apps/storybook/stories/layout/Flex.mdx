# Flex

Used to manage flex layouts. A flexible box layout component that makes it easy to align and distribute space among items in a container.

<Canvas of={FlexStories.Basic} />

## Usage

```tsx
<script>
	import { Flex } from "$saas/layout/flex";
</script>

<Flex gap={4}>
	<div>Item 1</div>
	<div>Item 2</div>
	<div>Item 3</div>
</Flex>
```

## Examples

### Direction

Use the `direction` prop to change the direction of the flex container.

### Justify
