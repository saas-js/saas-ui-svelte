# Grid

Used to manage grid layouts. The Grid component provides a simple way to create CSS grid layouts with responsive columns and rows.

<Canvas of={GridStories.Basic} />

## Usage

```tsx
<script>
	import { Grid, GridItem } from "$saas/layout/grid";
</script>

<Grid columns={3} gap={4}>
	<div>Item 1</div>
	<div>Item 2</div>
	<div>Item 3</div>
</Grid>
```

## Examples

### Spanning Columns and Rows

Use `GridItem` with `colSpan` and `rowSpan` props to span across columns or rows.

<Canvas of={GridStories.Spanning} />

### Custom Gap

Use `gapX` and `gapY` props to set different horizontal and vertical gaps.
