# Bleed

Used to break an element from the boundaries of its container. The Bleed component applies negative margins to allow content to extend beyond its parent's padding.

<Canvas of={BleedStories.Basic} />

## Usage

```tsx
<script>
	import { Bleed } from "$saas/layout/bleed";
</script>

<div class="p-8 border">
	<Bleed inline={8}>
		<div>This content bleeds horizontally</div>
	</Bleed>
</div>
```

## Examples

### Direction

Use `inlineStart`, `inlineEnd`, `blockStart`, and `blockEnd` props to bleed in specific directions.
