# Float

Used to anchor an element to the edge of a container.

<Canvas of={FloatStories.Basic} />

## Anatomy

```tsx
<script>
	import { Float } from "$saas/layout/float";
	import { Box } from "$saas/layout/box";
	import { Circle } from "$saas/layout/circle";
</script>

<Box class="relative h-20 w-20 bg-bg-subtle">
	<Float>
		<Circle size="sm" colour="red">3</Circle>
	</Float>
</Box>
```
