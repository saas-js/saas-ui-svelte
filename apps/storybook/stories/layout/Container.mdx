# Container

Used to constrain a content's width to the current breakpoint, while keeping it fluid. The default maxWidth is 8xl which maps to 90rem (1440px).

<Canvas of={ContainerStories.Basic} />

## Usage

```tsx
<script>
	import { Container } from "$saas/layout/container";
</script>

<Container>
	<div>Your content here</div>
</Container>
```

## Examples

### Sizes

Use the `maxW` prop to change the maximum width of the container.

<Canvas of={ContainerStories.Sizes} />

### Fluid

Use the `fluid` prop to make the container stretch to fill the width of its parent.
