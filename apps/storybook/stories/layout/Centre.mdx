# Centre

Used to centre its child within itself. The Centre component uses flexbox to centre its children both horizontally and vertically.

<Canvas of={CentreStories.Basic} />

## Usage

```tsx
<script>
	import { AbsoluteCentre, Centre, Square } from "$saas/layout/centre";
	import { Circle } from "$saas/layout/circle";
</script>

<Centre class="bg-bg-emphasized h-[100px]">
	This is the Centre
</Centre>
```

## Related Components

### Square

Square centres its child given the size (width and height).

<Canvas of={CentreStories.Square} />

### Circle

Circle centres its child given the size and creates a circle around it.

<Canvas of={CentreStories.Circle} />

### AbsoluteCentre

AbsoluteCentre centres relative to its parent using `position: absolute`.
