# Aspect Ratio

Used to embed responsive videos and maps, etc. The AspectRatio component is used to embed responsive content while maintaining a specific aspect ratio.

<Canvas of={AspectRatioStories.Basic} />

## Usage

```tsx
<script>
	import { AspectRatio } from "$saas/layout/aspect-ratio";
</script>

<AspectRatio ratio={16 / 9}>
	<iframe
		title="Video"
		src="https://www.youtube.com/embed/EHfx9LXzxpw"
		allowFullScreen
	/>
</AspectRatio>
```
