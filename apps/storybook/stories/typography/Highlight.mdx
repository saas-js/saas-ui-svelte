# Highlight

Used to highlight substrings of a text.

<Canvas of={HighlightStories.Basic} />

## Usage

```tsx
<script>
	import { Highlight } from "$saas/typography/highlight";
</script>

<Highlight query="Hello" text="Hello World" />
```

## Examples

### Multiple

Pass an array of strings to the `query` prop to highlight multiple substrings.

<Canvas of={HighlightStories.Multiple} />

### Custom style

Use the `variant` prop to customise the style of the highlighted text.

<Canvas of={HighlightStories.CustomStyle} />

### Search query

Use the highlight component to highlight search query results.

<Canvas of={HighlightStories.SearchQuery} />

### Mark class

Use the `markClass` prop to apply custom CSS classes to the highlighted text beyond the built-in variants.

<Canvas of={HighlightStories.MarkClass} />
